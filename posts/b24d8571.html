软件包管理器

pacman

用 C 语言编写的软件包管理器，安装需要加sudo

详细使用方法请参考pacman

命令 描述
sudo pacman -S 安装或升级单个软件包
sudo pacman -Syu 升级软件包和整个系统
sudo pacman -R 删除单个软件包保留依赖
sudo pacman -Rsu 删除软件包和所有依赖
sudo pacman -Scc 删除缓存中的全部文件
sudo pacman -Qs 查询已安装的软件包

yay

用 Go 编写的 AUR 助手，安装不需要加sudo。

详细使用方式请参考yay

命令 描述
yay -S 安装包
yay -Rns 删除包
yay -Qi 查询安装的版本
yay -Pc 打印系统统计信息
yay -Syu 升级所有已安装的包
yay -Yc 清除不需要的依赖项

使用方式：

比如我们想在Github里面搜索numpy，我们只需要以下三步：

  1. 快捷键Ctrl+空格启动Albert搜索框。
  2. 在搜索框中输入git numpy（注意中间有个空格）。
  3. 回车即可自动打开浏览器，跳转GitHub界面。
