HTML ：Hyper Text Markup Language(超文本标记语言)

目前网页中常用HTML5

W3C标准

W3C ：World Wide Web Consortium

成立于1994年，Web技术领域最权威和具影响力的国际中立性技术标准机构

W3C标准包括：

结构化标准语言（HTML XML）

表现标准语言（CSS）

行为标准（DOM(文档对象模型) ECMAScript(js)）
